# Ocean-awareness-web-application

This project is a front-end web application developed to raise awareness about ocean conservation and environmental protection.

## Features
- Informational pages about ocean conservation
- Feedback form with validation
- Star rating system
- Word counter functionality
- Responsive web design
- Team information page

## Technologies Used
- HTML
- CSS
- JavaScript
